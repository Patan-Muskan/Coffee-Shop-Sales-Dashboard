# ☕ Coffee Shop Sales Dashboard

## 📊 Overview
Power BI dashboard analyzing coffee shop sales 
across 3 NYC locations with 149K transactions.

## 🔑 Key Insights
- Peak orders at 8-10 AM
- Sales grew 85% Jan→Jun 2023
- 40% profit margin across all products

## 🛠️ Tools Used
- Power BI Desktop
- DAX
- Power Query
- Maven Analytics Dataset
